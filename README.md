# my-planner

# 🌸 StudentFlow — Personal Task Planner

A beautiful, feature-packed productivity planner built for busy students. No sign-up, no downloads, no subscription — just open it and go.

**Live site:** https://ahammer25.github.io/my-planner

---

## ✨ Features

- **Task management** — add tasks with categories, priority levels, and due dates
- **Subtasks** — break big assignments into smaller checkboxable steps
- **Task notes** — attach links, instructions, or extra context to any task
- **Today view** — flag tasks for today and focus on just those
- **Drag & drop** — reorder tasks by dragging them up or down
- **Pomodoro timer** — built-in 25/5/15 minute focus timer with sound alert
- **Upcoming deadlines** — see what's due soon at the top of the page
- **Task aging** — tasks that have been sitting too long are highlighted
- **Daily quote** — a fresh motivational quote every morning
- **Streak counter** — tracks how many days in a row you've used the app
- **Quick capture** — floating ⚡ button to jot things down instantly
- **Weekly summary** — see your progress across all categories
- **End of day celebration** — confetti when you finish everything for the day
- **Spotify embed** — study playlist built right into the sidebar
- **Export / Import** — download your tasks as a JSON backup anytime
- **All data saved locally** — nothing leaves your browser, fully private

---

## 🚀 How to use

1. Visit the live site link above
2. Add tasks using the input box — pick a category, priority, and due date
3. Click ☀️ on a task to add it to your **Today** focus list
4. Use the **Pomodoro timer** to work in focused sessions
5. Click **⬇ Export** in the sidebar regularly to back up your tasks
6. If you're switching devices, export on the old device and import on the new one

---

## 💾 Saving your data

Your tasks are saved in your browser's **localStorage** — they survive refreshes and restarts but are tied to this browser. To avoid losing data:

- Use **Export** to download a backup `.json` file
- Use **Import** to restore from a backup on any device
